# 📊 대한민국 시도별 자살률 결정 요인 분석 (Korea Suicide Rate Analysis)
대한민국의 사회적 이슈인 자살률에 영향을 미치는 주요 요인을 데이터로 분석하고, 지역별 격차를 시각화한 프로젝트입니다. 파이썬을 활용한 통계적 검증과 Tableau를 이용한 인터랙티브 대시보드 제작을 포함하고 있습니다.

## 📌 프로젝트 개요

- **분석 목적**: 인구 구조(고령화), 경제적 여건(소득, 기초생활수급자 수, 고용률, 실업률 등), 정신건강 및 정신건강 관련 인프라 등 다양한 변수가 지역별 자살률에 미치는 영향을 분석하여 유의미한 상관관계를 도출함.

- **데이터 소스**: 통계청(KOSIS) 및 공공데이터포털 기반 시도별 통계 데이터

- **주요 기술**: Python (Pandas, Statsmodels, Matplotlib, Seaborn), Tableau

## 🔍 주요 분석 내용 (Python 기반)

### 1. 상관관계 분석 (Correlation Analysis)

자살률과 각 사회 지표 간의 상관관계를 분석한 결과, 다음과 같은 유의미한 결과가 도출되었습니다.


- **고령인구비율**: 자살률과 가장 높은 정(+)의 상관관계(0.52)를 보임 (p < 0.001).

- **개인 소득**: 자살률과 부(-)의 상관관계(-0.44)를 나타내며, 소득 수준이 낮을수록 자살률이 높은 경향을 보임.

- **기타**: 정신건강 관련 기관 수(-0.40), 기초생활수급자 수(-0.35) 또한 자살률과 유의미한 상관관계를 가짐.

- **무관한 지표**: 실업률, 스트레스 인지율 등은 본 데이터셋에서 통계적으로 유의미한 상관관계가 낮게 나타남.

---------------------------------------------------------------------------------------------------------------------------------------------------------------
### 2. 다중회귀분석 (Multiple Regression Analysis)

단순한 변수 선택을 넘어, 전체 지표를 검토하고 모델을 고도화하는 과정을 거쳤습니다.


**✔️ Step 1. 전체 변수 투입 분석 (Exploratory Analysis)**

- 분석 내용: 고령인구비율, 개인 소득, 정신건강기관 수, 기초생활수급자 수, 실업률, 스트레스 인지율 등 수집된 모든 독립 변수를 투입하여 자살률에 대한 포괄적 회귀 분석을 실시했습니다.

- 분석 결과: 대다수의 변수들이 통계적 유의수준($p > 0.05$)을 만족하지 못하는 것으로 나타났습니다.

- 의미 파악: 이는 개별 지표들이 자살률과 단순 상관관계는 있으나, 서로 간의 다중공선성(Multicollinearity)이나 데이터의 노이즈로 인해 독립적인 영향력을 증명하기 어려움을 의미합니다. 이 과정을 통해 데이터의 복잡성을 확인하고 변수 정제(Variable Selection)의 필요성을 도출했습니다.


**✔️ Step 2. 주요 변수 기반 최종 모델 (Refined Model)**

- 변수 선정: 탐색적 분석 결과를 바탕으로 이론적 연관성이 가장 높고 영향력이 컸던 세 변수(고령화, 소득, 수급자 수)를 중심으로 모델을 재구성했습니다.

- 분석 결과 : 세 변수(고령화, 소득, 수급자 수) 자살률의 약 37% 정도를 설명합니다. 상관 분석에서는 통계적으로 유의했으나 회귀 분석에서는 통계적으로 유의하지 않아 확실한 설명 변수는 없다고 볼 수 있습니다.

- 종합 해석: 

• 상관분석에서는 유의미했던 지표들이 회귀 모델에서 비유의적으로 나타나는 것은, 특정 지표 하나가 자살률을 결정짓기보다 여러 사회적 요인이 복합적으로 얽혀 있음을 시사합니다.

• 특히 **"분석되지 않은 숨은 변수(Residuals)"** 의 비중이 높다는 점을 확인하였으며, 이는 향후 정신건강 서비스 접근성이나 정서적 지원 인프라 등 정성적 지표의 데이터화가 필요함을 정량적으로 증명한 성과입니다.
